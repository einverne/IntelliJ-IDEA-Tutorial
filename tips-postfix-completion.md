# Postfix completion

可能听名字比较奇怪，但是看演示

![postfix completion](images/postfix-completion.gif)

对于 Java 有如下

- fori 正序遍历collection
- forr 逆序
- switch
- sout 打印表达式 System.out.println
- null 检查是否为null
- notnull 检查不为null
- try 添加 try
